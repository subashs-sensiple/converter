# Steps to build the exectable jar file and run it.
Step1: mvn clean package -e
Step2: After executing Step1 "converter-0.0.1-SNAPSHOT-jar-with-dependencies.jar" will be generated in target folder.
Step3: Open CLI run this command "java -jar <SPECIFY_JAR_FILE_PATH>/converter-0.0.1-SNAPSHOT-jar-with-dependencies.jar"
Step4: It will as you to enter input & output file path. (ex: Input file path: D:\CSW\Input\case1.json & Output file path: D:\CSW\Output\case1.xml)
Step5: You will get the result.


# Steps to run main program.
Step1: Goto the pacakge src/main/java/com/csw/converter
Step2: Run Application.java
Step3: It will as you to enter input & output file path. (ex: Input file path: D:\CSW\Input\case1.json & Output file path: D:\CSW\Output\case1.xml)
Step4: You will get the result.
